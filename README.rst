###############################
How do I get set up?
###############################

Instale usando PyPI:

    pip install django_invisible_recaptcha_admin

Add the app ``django_invisible_recaptcha_admin`` to ``settings.py``



    INSTALLED_APPS = (
        'django_invisible_recaptcha_admin',
        
        'nocaptcha_recaptcha',

        ...
    )
